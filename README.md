# 🏆 중등 2030수업나눔한마당 백일장

전남교육청 중등 2030수업나눔한마당 행사용 백일장 웹페이지입니다.

## 주요 기능

- ✍️ **백일장 입력**: 학생들이 글을 작성하고 제출
- - 🎨 **전남교육청 테마**: 교육청 공식 색상 및 디자인 적용
  - - 📱 **반응형 디자인**: 모바일/태블릿 지원
   
    - ## 기술 스택
   
    - - **Frontend**: HTML, CSS (TailwindCSS)
      - - **Fonts**: Noto Sans KR, Jua
